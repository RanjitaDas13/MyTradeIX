Steps:-
1. Download MyTask folder entirely to your local system.
2. open command prompt
3. check in maven is installed using "mvn -v" command
4. If maven is not installed, install maven
5. Go to the folder(MyTask) in command prompt
6. run "mvn clean test"
7. You will see the tests run which shows employee created, salary updated and employee deleted in the same screen.
8. To rerun the tests you can reuse "mvn test" command in command prompt
9. Verify the logs generated in MyTask/logs/log.log file which keeps on getting updated with everytime you run the tests.
